# Precision PDF — Final Design Delivery Pack

This folder collects the current design materials for the PDF Page Remover project.

## Contents
- `01_stitch_prompt.md` — the stitched optimization prompt
- `02_design_handoff.md` — component-level handoff
- `03_frontend_spec.md` — page structure and content spec
- `04_dev_task_list.md` — implementation checklist for front-end
- `05_frontend_handoff.md` — final homepage + workspace handoff
- `06_frontend_handoff_short.md` — short version for direct front-end use

## Recommended use
1. Read `README.md` first for the overall map.
2. Use `01_stitch_prompt.md` when running Stitch.
3. Share `06_frontend_handoff_short.md` with frontend for the fastest implementation brief.
4. Use `05_frontend_handoff.md` when you want the fuller homepage + workspace handoff.
5. Use `02_design_handoff.md` for structure and states.
6. Use `03_frontend_spec.md` for implementation details.
7. Use `04_dev_task_list.md` to track build priority.

## Product boundary
Single-function PDF page remover only.
Do not expand into merge/split/compress/OCR/sign/convert or a PDF suite.
